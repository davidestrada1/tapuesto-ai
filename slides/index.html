The Challenge

01 The Problem

  • People love showing off their expertise 🏆
  • Friendly bets are fun but untracked 💰
  • No way to build a prediction reputation 📊
  • No gamified comparison with peers 👥

The gap: Social media shows opinions, not accuracy. There's no "prediction credit score."

Core Technology

08 AI Integration

Model: gemini-3-flash-preview

geminiService.ts 
const response = await ai.models.generateContent({
    model: "gemini-3-flash-preview",
    contents: `Analiza el tema: "${topic}".
        1. Busca información reciente.
        2. Proponer PREGUNTA binaria clara.
        3. Dar razonamiento.`,
    config: {
        tools: [{ googleSearch: {} }],
    },
});
  • Contextual Analysis — turns vague ideas into strict YES/NO
  • Google Search Grounding — validates relevance with live web data
  • Structured Output — returns formatted question + reasoning + sources

Differentiation

11 What Makes This Unique

  1. AI-first bet creation — Gemini + Google Search validates and formats every bet
  2. Dual-language from day one — not an afterthought, built for ES/EN markets
  3. Reputation is the product — ELO, hit rate, payer score, leaderboards
  4. Flexible stakes — cash, drinks, chores, anything is wager-able
  5. Proof-based settlement — photo, video, link, voucher evidence system
